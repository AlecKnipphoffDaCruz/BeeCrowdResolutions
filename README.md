# BeeCrowd Solutions & Python Learning

A personal repository for practicing Python and Java through BeeCrowd competitive programming problems and syntax exercises.

## Goals

- Solve BeeCrowd algorithmic problems in Python and Java
- Practice and reinforce Python fundamentals
- Explore Python standard library features (file I/O, JSON, etc.)

## Repository Structure

```
BeeCrowdSolutions/
├── java/                        # Java solutions to BeeCrowd problems
│   ├── BEE1016.java             # Distance/speed calculation
│   └── BEE1018.java             # Currency denomination (Brazilian Real)
└── python/
    ├── beecrowd/                # Python solutions to BeeCrowd problems
    │   └── BEE1016.py
    ├── syntax/                  # Core Python syntax practice
    │   └── __practice__.py
    ├── working_with_files/      # File I/O exercises
    │   ├── __practice2__.py
    │   └── files/               # Sample .txt data files
    └── working_with_json/       # JSON manipulation exercises
        ├── __practice3__.py
        └── jsons/               # Sample .json data files
```

## Content Overview

### BeeCrowd Problems

| Problem | Description | Languages |
|---------|-------------|-----------|
| BEE1016 | Distance calculation between two cars at different speeds | Python, Java |
| BEE1018 | Break a value into Brazilian Real denominations | Java |

### Python Practice Modules

#### `syntax/__practice__.py`
Fundamentals of Python syntax:
- String formatting and user input
- Conditionals (age verification)
- List operations and filtering
- Dictionary creation and iteration
- List comprehensions vs traditional loops
- Array transformations

#### `working_with_files/__practice2__.py`
File I/O operations:
- Reading files line by line
- Parsing and storing data from text files
- Computing averages from grade files
- Using `map` and `lambda` for file data processing
- Writing processed data back to files

#### `working_with_json/__practice3__.py`
JSON handling with the `json` module:
- Reading and parsing JSON files
- Writing Python dicts to JSON
- Merging multiple JSON sources
- Managing a product catalog with stock updates (simulated shopping system)

## Languages & Tools

- **Python 3** — primary language
- **Java** — used for BeeCrowd problem solutions
- **BeeCrowd** — online judge platform for algorithmic challenges

## About BeeCrowd

[BeeCrowd](https://www.beecrowd.com.br) (formerly URI Online Judge) is a Brazilian competitive programming platform with problems ranging from beginner math to advanced algorithms.
